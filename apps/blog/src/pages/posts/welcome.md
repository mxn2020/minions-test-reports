# Welcome to Minions Test-reports

We're excited to introduce **Minions Test-reports** — Aggregated results, regression diffs, quality trends, and coverage maps.

## What's Included

- **TypeScript SDK**: `npm install @minions-test-reports/sdk`
- **Python SDK**: `pip install minions-test-reports`
- **CLI**: `npm install -g @minions-test-reports/cli`

## Getting Started

Check out the [documentation](https://test-reports.minions.help) for a full guide.
