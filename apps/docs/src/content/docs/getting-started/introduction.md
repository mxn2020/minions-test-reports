# Minions Test-reports

Aggregated results, regression diffs, quality trends, and coverage maps

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-test-reports/sdk`) — Core library for Node.js
- **Python SDK** (`minions-test-reports`) — Python bindings
- **CLI Tool** (`@minions-test-reports/cli`) — Command-line interface
- **Web App** — Interactive playground at [test-reports.minions.wtf](https://test-reports.minions.wtf)
